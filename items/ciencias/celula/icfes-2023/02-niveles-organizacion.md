# Niveles de Organización – Prueba de Imágenes

**RESPONDE LAS PREGUNTAS 1 A 3 DE ACUERDO CON LA SIGUIENTE INFORMACIÓN:**

Tu cuerpo tiene una organización jerárquica: células forman tejidos, tejidos forman órganos, órganos forman sistemas, y todos los sistemas forman el organismo completo.

![la-jerarquia-de-la-vida](https://cdn.ediprofe.com/img/ciencias/a38z-la-jerarquia-de-la-vida.webp)

---

## 1.

Observa la imagen del sistema muscular. ¿Cuál es el orden correcto de organización desde lo más pequeño a lo más grande?

![el-camino-del-movimiento-sistema-muscular](https://cdn.ediprofe.com/img/ciencias/xegd-el-camino-del-movimiento-sistema-muscular.webp)

- A. Sistema → Órgano → Tejido → Célula
- B. Célula → Tejido → Órgano → Sistema
- C. Tejido → Célula → Sistema → Órgano
- D. Órgano → Sistema → Célula → Tejido

<details>
<summary>✅ Respuesta</summary>

**Respuesta: B**

El orden correcto es de menor a mayor complejidad:
- Célula (miocito) → Tejido (muscular) → Órgano (bíceps) → Sistema (muscular)

</details>

## 2.

Según la imagen del sistema nervioso, ¿qué tipo de célula es la unidad básica de este sistema?

![el-camino-de-control-sistema-nervioso](https://cdn.ediprofe.com/img/ciencias/1dx5-el-camino-de-control-sistema-nervioso.webp)

- A. Miocito
- B. Enterocito
- C. Neurona
- D. Glóbulo rojo

<details>
<summary>✅ Respuesta</summary>

**Respuesta: C**

La neurona es la célula especializada del sistema nervioso, encargada de transmitir impulsos eléctricos.

</details>

## 3.

En la imagen de integración del organismo, ¿por qué el organismo representa el nivel más alto?

![la-integracion-fina-organismo](https://cdn.ediprofe.com/img/ciencias/16js-la-integracion-fina-organismo.webp)

- A. Porque es el más pequeño
- B. Porque solo tiene un sistema
- C. Porque integra todos los sistemas trabajando juntos
- D. Porque no tiene células

<details>
<summary>✅ Respuesta</summary>

**Respuesta: C**

El organismo es el nivel máximo porque representa la integración de TODOS los sistemas (muscular, nervioso, digestivo, etc.) funcionando de manera coordinada.

</details>
